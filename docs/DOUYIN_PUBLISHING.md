# 抖音虚拟创作平台发布说明

目标地址：https://vcreate.douyin.com/

## 当前工程形态

本项目是纯静态 H5：

- 无后端服务依赖
- 无数据库依赖
- 无运行时 AI 生成依赖
- 入口：`index.html`
- 构建目录：`dist/`
- 推荐平台：移动浏览器 / 抖音互动空间 H5 容器

## 构建命令

```bash
npm install
npm run build
```

构建完成后检查：

```text
dist/index.html
dist/assets/*.js
dist/assets/*.css
```

## 平台侧待确认

以下内容需要在抖音虚拟创作平台账号内确认：

1. H5 上传方式：直接上传 `dist/`、ZIP 包，或填写已托管 URL。
2. 分享 API：是否需要接入平台 JS SDK。
3. 互动空间 API：是否有指定初始化、上报或退出方法。
4. 审核资料：项目名、简介、封面图、玩法截图、团队信息。
5. 资源限制：包体大小、首屏加载、远程资源域名白名单。

## 关于“所有人可以 push”

GitHub 公共仓库支持所有人访问、fork 和提交 Pull Request，但不支持“互联网所有人直接 push 到主仓库”的公开开关。直接 push 权限只能授予指定协作者、团队或组织成员。
